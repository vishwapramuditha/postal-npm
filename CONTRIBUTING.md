# Contributing to sri-lanka-postal-codes

Thank you for considering contributing to this project! We welcome your help to make this the best resource for Sri Lankan postal code data.

## How can I help?

1.  **Report Bugs**: If you find an incorrect postal code or city details, please open an issue.
2.  **Add Missing Data**: If a city is missing, feel free to add it.
3.  **Improve Code**: Optimizations and bug fixes are always welcome.

## Submitting specific changes

1.  Fork the repository.
2.  Create a new branch for your feature or fix.
3.  Make your changes.
4.  Run tests to ensure everything is working:
    ```bash
    npm test
    ```
5.  Submit a pull request.

## Data Guidelines

When adding or modifying data in `src/data.json`, please ensure:
-   **Accuracy**: Verify the postal code from an official source if possible.
-   **Format**: Maintain the existing JSON structure.
-   **Languages**: If possible, provide English, Sinhala, and Tamil names.

Thank you!
